## Key Legend

| Key | Action |
| --- | :----: |
| Tab | Moves across a tab |
| Shift-Tab | Moves back a tab |
| k | go up |
| j | go down | 
| h | go left/exit out of |
| l | go right/enter into |
| ctrl-c | quit |
| q | quit |
